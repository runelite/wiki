# Minimap Configuration

## Settings

### Hide minimap

Allows hiding the minimap while using the `Resizable` game client layout.

![resizable](img/minimap/minimap_resizable.png)

### Zoom

Enables zooming the minimap

### Minimap dot colors

Customizes the color of minimap dots.

![minimap dot settings](img/minimap/minimap_dots_settings.png)

![minimapdots](img/minimap/minimap_dots_example.png)
